<h2>EL(Expression Language)</h2>
: EL이란, 표현식 또는 액션 태그를 대신해서 값을 표현하는 언어입니다.  
```html
<%= value %>     ---->     ${ value }
```
<h4>EL연산자</h4>
- 산술: +, -, *, /, %  
- 관계형: ==, !=, <, >, <=, >=  
- 조건: a? b : c  
- 논리: &&, ||  
<h4>Ex</h4>
```html
  ${10}<br/>
	${99.99}<br/>
	${"ABC"}<br/>
	${true}<br/>
  ${ 1+2 }<br/>
	${ 1-2 }<br/>
	${ 1*2 }<br/>
	${ 1/2 }<br/>
	${ 1>2 }<br/>
	${ 1<2 }<br/>
	${ (1>2)? 1 : 2 }<br/>
	${ (1>2) || (1<2) }<br/>
```
