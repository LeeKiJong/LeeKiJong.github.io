<h2>Servlet</h2>
- 동적 웹어플리케이션 컴포넌트  
- Java 확장자.  
- 클라이언트 요청에 동적으로 작동하고, 응답은 html을 이용.  
- java thread를 이용하여 동작.  
- MVC패턴에서 Controller로 이용됨.
